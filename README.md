# Анализ рынка недвижимости СПб: SQL-аналитика и интерактивный дашборд в Yandex DataLens.
## 🛠 Технологический стек
*   **Хранение данных:** PostgreSQL (схемы, индексы, агрегации).
*   **Аналитика:** SQL (CTE, оконные функции, сегментация).
*   **Визуализация:** Yandex DataLens.

### 📊 Визуализация в Yandex DataLens - https://datalens.yandex/v2s2pfy3mb9eh
В проекте реализован дашборд, который позволяет в реальном времени отслеживать:
* Динамику стоимости м² по месяцам.
* Среднюю цену/кол-во квартир/ср.кол-во дней в продаже по районам СПб и ЛО.
* ТОП-5 в зависимости от разных характеристик объекта.
