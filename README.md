# Анализ рынка недвижимости СПб: хранение в PostgreSQL, SQL-аналитика и интерактивный дашборд в Yandex DataLens.
## 🛠 Технологический стек
*   **Хранение данных:** PostgreSQL (схемы, индексы, агрегации).
*   **Аналитика:** SQL (CTE, оконные функции, сегментация).
*   **Визуализация:** Yandex DataLens.

## 🚀 Ключевые инсайты и рекомендации

### 1. Стратегия «Быстрые продажи»
Для сокращения цикла сделки выявлены оптимальные параметры ликвидных объектов:
 Параметр | Санкт-Петербург | Ленинградская область |
| :--- | :--- | :--- |
| **Цена за м²** | **108 000 руб.** (на 3% ниже рынка) | **72 000 руб.** (рыночная цена) |
| **Тип объекта** | 2-к. кв (54.6 м²) или Студия | 2-к. кв (49 м²) |
| **Локация** | Масс-маркет без парков | С обязательным парком/водоемом |

### 2. Территориальная стратегия
Рынки СПб и ЛО требуют **раздельного подхода**:
*   **Масс-маркет (фокус):** Мурино, Шушары, Парголово, Всеволожск, Кудрово.
*   **Премиум (высокая окупаемость):** Пушкин, Сестрорецк.
*   **Бюджетный сегмент:** Красное Село, Ломоносов, Янино-1.

> **Заметка по сегментам:** Рынок студий/апартаментов сейчас переоценен и малочислен. Рекомендуется мониторинг новых строек для оценки потенциала ниши.

### 3. Маркетинговый календарь
*   **Усиление охватов:** Февраль, Октябрь, Ноябрь.
*   **Максимальная маржа:** Декабрь, Январь, Март (пиковые цены за м²).
*   **Стимулирование:** В Мае закладывать небольшие скидки для ускорения продаж.

### 📊 Визуализация в Yandex DataLens - https://datalens.yandex/v2s2pfy3mb9eh
В проекте реализован дашборд, который позволяет в реальном времени отслеживать:

Средняя цена/кол-во квартир/ср.кол-во дней на продаже в динамике по районам СПб и ЛО.
